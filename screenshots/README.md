# Screenshots

This folder contains screenshots for the GitHub SSH Jenkins tutorial.

## Screenshot Naming Convention

Please name your screenshots following this pattern:
- `step1-ssh-keygen.png` - SSH key generation
- `step2-github-ssh-settings.png` - GitHub SSH settings page
- `step2-add-ssh-key.png` - Adding SSH key to GitHub
- `step3-jenkins-credentials.png` - Jenkins credentials page
- `step3-add-credential.png` - Adding credential in Jenkins
- `step4-jenkins-job-config.png` - Jenkins job configuration
- `step4-ssh-url.png` - SSH URL in repository settings
- `step5-build-success.png` - Successful build
- `step5-workspace-files.png` - Files in Jenkins workspace

## Adding Screenshots

1. Take screenshots at each step
2. Save them in this folder with descriptive names
3. Update the main README.md to reference these screenshots
