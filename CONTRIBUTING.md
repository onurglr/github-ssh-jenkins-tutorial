# Contributing

Thank you for your interest in contributing to this guide!

## How to Contribute

1. Fork this repository
2. Create a feature branch (`git checkout -b feature/improvement`)
3. Make your changes
4. Commit your changes (`git commit -m 'Add some improvement'`)
5. Push to the branch (`git push origin feature/improvement`)
6. Open a Pull Request

## Guidelines

- Keep the guide clear and step-by-step
- Test the instructions on a fresh setup if possible
- Update screenshots if UI changes significantly
- Maintain consistency with existing formatting
